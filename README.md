# Calculadora em C

Este programa é uma calculadora simples em **C**, utilizando **funções** para cada operação matemática.  
Com implementação de estrutura de repetição e condição.
---

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão (com tratamento de divisão por zero)
- Menu interativo
